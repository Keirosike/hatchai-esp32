HatchAI

Manage your smart hatch system

HatchAI is an ESP32-based local incubator monitoring and control system. It helps authorized users monitor incubation conditions, manage hatchery data, and access device responses directly through a local dashboard.

Secure Access

The login page protects the local dashboard so only permitted users can access the incubator system.

Edge Processing

Data is processed locally on the ESP32, reducing delay and supporting operation without depending on a cloud server.
